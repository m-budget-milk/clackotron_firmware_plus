# Clackotron 2020 - Firmware & Webinterface
This repository is a fork of the original Clackotron 2000 firmware and web interface.
The goal of this project is to expand the platform so SBB split-flap displays can be used as realistic train departure and passenger information boards. It builds on the excellent Clackotron hardware and UI, while adding functionality inspired by Harkles’ Fallblatt modules.

Just to be clear, i am NOT a programmer. Most of the programming was done by AI.
